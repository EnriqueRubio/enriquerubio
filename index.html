PORTFOLIO

Bienvenido a mi portfolio, explora una colección diversa de trabajos que abarcan diseño web, desarrollo, y otros proyectos que he tenido el privilegio de llevar a cabo. Recuerda visitar mi perfil de GitHub para ver todos mis proyectos.

Proyecto Python: Mapeo de Rutas Urbanas con XML

En este proyecto, he desarrollado una aplicación en Python que transforma datos XML de calles y cruces de una ciudad en una representación de grafos compuesta por nodos y aristas. Esta estructura de datos se convierte en la base para calcular rutas eficientes y precisas dentro de la ciudad.

Proyecto React: Aplicacion Web de Notas

Junto a mis compañeros, hemos construido una aplicación web de notas potenciada por tecnologías modernas. Utilizando React para la interfaz de usuario dinámica y ágil, MongoDB como base de datos flexible y Ruby on Rails para el backend robusto, la aplicación ofrece una experiencia fluida de gestión de notas.

Proyecto WPF: Pokemon

Esto es un proyecto desarrollado con WPF (Windows Presentation Foundation), en el cual he creado una experiencia interactiva centrada en el mundo Pokémon. La aplicación presenta animaciones cautivadoras que dan vida a los personajes y una serie de acciones con las que puedes interactuar con el pokemon.
